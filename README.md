# Portafolio y guía comercial PBS

Página web local y estática para consultar el portafolio de programas de Pacífico Business School y apoyar al equipo comercial.

## Uso local

Abre `index.html` en un navegador moderno. No requiere instalación, servidor ni credenciales.

## Fuente de información

El contenido académico y comercial fue elaborado exclusivamente a partir de los brochures suministrados para Business Data Analytics, Liderazgo Estratégico y Adaptación al Cambio, y Evaluación Privada de Proyectos & Project Finance.

## Publicación

El proyecto está preparado para desplegarse como sitio estático en Vercel.
